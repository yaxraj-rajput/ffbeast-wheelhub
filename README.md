# FFBeast WheelHub

A web-based configurator for the FFBeast wheel, powered by the [FFBeast Wheel WebHID API](https://github.com/shubham0x13/ffbeast-wheel-webhid-api). Configure your FFBeast racing wheel directly from your browser, no installation required.

## Web-Based Configuration

FFBeast WheelHub runs entirely in your web browser using the **WebHID API**, eliminating the need for local software installation. Simply open the app in a supported browser to detect and configure your wheel instantly.

### Browser Compatibility

| Browser            | Support |
| :----------------- | :-----: |
| **Chrome**         |   ✅    |
| **Edge**           |   ✅    |
| **Opera**          |   ✅    |
| **Brave**          |   ✅    |
| **Chromium-based** |   ✅    |

> **Note:** Firefox and Safari are **not** supported as they do not currently implement the WebHID API.

## Features

- **Auto-Detection:** Automatically detects connected FFBeast wheels.
- **Multi-Profile Support:** Create and manage multiple configuration profiles for different games or driving styles.
- **Wheel Preview:** Visual representation of your wheel and current settings.
- **Torque Visualization:** Real-time torque bar display during operation.
- **Contextual Help:** Built-in tooltips and information guides for all settings.
- **Theme Support:** Multiple theme options for a personalized appearance.
